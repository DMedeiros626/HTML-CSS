Codigo-fonte

O comando document.getElementById9('teste') é escrito em JavaScrip

Foi usado a tag "code" para deixar a fonte mono espaçada no comando acima.

Para escrever um codigo de linguagem de maneira organizada é só usar a tag pre antes da tag code.

Exemplo


        
            num = int(input('Digite um numero'))
            if num % 2 == 0:
                 print(f'0 numero {num} é PAR')
            else:
                 print(f'0 numero {num} é IMPAR')
            print('Fim do programa')
